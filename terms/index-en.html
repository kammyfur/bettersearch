In which conditions use BetterSearch?

This page details in which conditions you can use BetterSearch, such as at school, commercially, and more...

Individual usage

If you're an individual, feel free to use BetterSearch, and its API (but follow the terms of use of the API).

Educational usage

If you work for a school and want to deploy BetterSearch as default search engine, or promote it, you need to ask us by sending email to us at minteck.projects+edu@gmail.com.

Professional usage

If you want to use BetterSearch at your work, feel free to do so, but you should send email to us at minteck.projects+pro@gmail.com.

Commercial usage

If you want to sell products that are using BetterSearch, you musn't add additional fees, and give choice to end user to disable it if wanted.

End User License Agreement

BetterSearch website is released under proprietary license, add-ons are under MIT license and apps under GNU-LGPL3 license.

The website's proprietary license forbid access to the source code, and every software working like BetterSearch may be considered as plagiarism, or even counterfeit if a BetterSearch term appears in it.

Prohibitions

By using BetterSearch, you can't...

  • use it for illegal activites
  • bypass software operation
  • use a software that changes BetterSearch appearance or operation
  • abuse BetterSearch and/or BetterSearch:cloud
  • break the law (French law)
